# Automation Playbook — Persian Tools

این چک‌لیست برای اجرای منظم خودکارسازی پروژه است و با کانفیگ Codex هم‌راستاست.

**پیش‌نیازها**

1. `pnpm` نصب باشد.
2. متغیرهای محیطی لازم برای دیتابیس و سرویس‌ها تنظیم باشد.
3. مسیر پروژه: `/home/dev/My_Project/persian-tools`.

**چک‌لیست اجرایی**

1. استانداردها را از `docs/project-standards.md` مرور کن.
2. معماری را از `docs/architecture.md` مرور کن.
3. تغییرات کد را اعمال کن و وابستگی خارجی در runtime اضافه نکن.
4. تست‌ها را اجرا کن:
5. `pnpm vitest --run`
6. `pnpm lint`
7. `pnpm typecheck`
8. در صورت نیاز تست E2E:
9. `pnpm test:e2e`
10. خطاها را برطرف کن و مجدد تست بگیر.
11. در صورت تغییر دیتابیس، `scripts/db/schema.sql` را هم‌راستا کن.
12. گزارش تغییرات و وضعیت تست‌ها را آماده کن.
13. کامیت با Conventional Commits انجام بده.
14. Push به ریموت در صورت فعال بودن فرآیند انتشار.

**یادداشت‌ها**

1. پوشه `docs` در `.gitignore` است مگر خلاف آن تنظیم شود.
2. UI باید فارسی و RTL باشد و از فونت IRANSansX استفاده کند.
