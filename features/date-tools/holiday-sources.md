# منابع تعطیلات قمری

> آخرین به‌روزرسانی: 2026-02-03

این فایل برای ثبت منبع رسمی تعطیلات قمری استفاده می‌شود تا داده‌ها در هر سال
با «تقویم رسمی کشور» همسان شوند.

## منبع رسمی

- استخراج تقویم رسمی توسط «مرکز تقویم مؤسسه ژئوفیزیک دانشگاه تهران»
- مناسبت‌های مصوب توسط «شورای فرهنگ عمومی»

## نکته مهم

تاریخ‌های قمری بر پایه تقویم محاسباتی هستند و ممکن است با رؤیت هلال یک روز
اختلاف داشته باشند. در صورت انتشار نسخه جدید تقویم رسمی، این داده‌ها باید
به‌روز شوند.
