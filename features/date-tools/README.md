# Date Tools

این ماژول حاوی منطق سمت کلاینت برای تبدیل تاریخ‌های شمسی و میلادی، محاسبه سن، تفکیک اختلاف تاریخ و تحلیل روز هفته است. هدف، ارائه ابزارهای دقیق تاریخی بدون وابستگی به API خارجی و کاملاً در مرورگر کار کردن است.

## آنچه این ماژول انجام می‌دهد

- تبدیل دوطرفه بین تاریخ شمسی و میلادی (براساس الگوریتم Jalaali).
- بررسی اعتبار تاریخ‌ها و محاسبه تعداد روزها، ماه‌ها و سال‌ها بین دو نقطه زمانی.
- محاسبه سن دقیق و گردش بدون وابستگی به محاسبات سروری.
- یافتن نام روز هفته و جابجایی به جلو یا عقب به تعداد روز مشخص.
- ارائه `DateParts`, `CalendarType` و توابع کمکی مانند `normalizeToGregorian`, `differenceInYmd` و `addDays`.

## ساختار

- `date-tools.logic.ts`: مجموعه توابع محاسباتی با استفاده از تاریخ‌های UTC برای جلوگیری از اثر منطقه زمانی.
- `date-tools.logic.test.ts`: تست‌های واحد با Vitest برای اطمینان از تبدیل درست، اعتبارسنجی و اختلاف‌ها.
- `holidays.ts`: تعطیلات ثابت تقویم شمسی (آفلاین) برای نمایش سریع در UI.
- `components/features/date-tools/DateToolsPage.tsx`: صفحه‌ای با فرم‌های ورودی و کارت‌هایی برای هر ابزار.
- `app/(tools)/date-tools/page.tsx`: Route فایل با metadata مناسب سئو.

## توسعه

1. تغییرات را روی شاخه خود اعمال کنید.
2. `pnpm install` برای نصب وابسته‌ها (اگر قبلاً نصب نشده).
3. برای اجرای تست‌های این ماژول: `pnpm vitest run features/date-tools/date-tools.logic.test.ts`.

## نکات

- همهٔ محاسبات به‌صورت UTC انجام می‌شوند تا اثر ساعات تابستانی یا منطقه زمانی از بین برود.
- تاریخ‌های ورودی باید در قالب `YYYY/MM/DD` با ارقام فارسی یا انگلیسی باشند.
- خطاها در صفحه نمایش داده می‌شوند تا کاربر بداند چه تاریخی نامعتبر است.
