# استراتژی درآمدزایی (با محوریت حریم خصوصی)

> آخرین به‌روزرسانی: 2026-02-06

## اصول کلیدی

- **رضایت کاربر قبل از هر تبلیغ/تحلیل‌گر** (Opt‑in).
- **بدون وابستگی خارجی در runtime**؛ همه اسکریپت‌ها self‑host.
- **شفافیت کامل** درباره نوع داده‌های جمع‌آوری‌شده.
- **حداقل‌گرایی داده**: فقط متریک‌های ضروری برای بهبود محصول.

## مدل‌های مجاز

1. **حمایت/اسپانسر مستقیم** (بدون رهگیری شخصی).
2. **تبلیغات استاتیک محلی** پس از رضایت کاربر.
3. **اشتراک اختیاری** برای قابلیت‌های پیشرفته (در صورت وجود backend).

## ممنوعیت‌ها

- هرگونه رهگیری کاربر بدون رضایت.
- CDN یا اسکریپت ثالث در مسیر runtime.

## الزامات اجرایی

- بنر رضایت تبلیغات قبل از رندر `AdSlot`.
- تحلیل‌گر داخلی فقط با `NEXT_PUBLIC_ANALYTICS_ID` فعال شود.
- اگر `ANALYTICS_INGEST_SECRET` تنظیم شد، دسترسی ingest محدود شود.
