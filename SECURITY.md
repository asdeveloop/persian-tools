# Security Policy

## Supported Versions

We support the latest `main` branch and the most recent tagged release.

## Reporting a Vulnerability

Please report security issues privately via email at security@persian-tools.ir.
Include a detailed description, steps to reproduce, and any proof of concept.
We will acknowledge your report within 72 hours.

## Coordinated Disclosure

We follow coordinated disclosure practices. Do not open public issues for security bugs.
